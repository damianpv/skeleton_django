# skeleton_django# 

Basic skeleton for Django.


## Overview ##

### Prerequisites ###

* Python >= 2.7+
* PIP
* virtualenv/virtualenvwrapper (optional)

## Installation ##
    
    cd {{ project_name }}
    pip install -r requirements.txt

### Sync database ###

    manage.py syncdb
    
### Running ###

    manage.py runserver

Open browser to http://127.0.0.1:8000 or http://localhost:8000
